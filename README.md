# Celebrate for Change ‚ö°

**Celebrate for Change** is a premium, Tesla-inspired platform designed to transform personal celebrations into real-world social impact. Built for students, visionaries, and change-makers, the platform redirects the generosity of birthdays and special events toward high-impact, verified charities globally.


## üé® Design Philosophy: 

The platform features a high-tech, futuristic design system inspired by minimalist luxury and performance:

-   **Permanent Dark Mode**: A high-contrast, obsidian-black interface that ensures premium look and readability in all conditions.
-   **Glassmorphism**: Elegant, semi-transparent components with subtle blur and edge-glow effects.
-   **Vibrant Accents**: A signature emerald-to-teal gradient that signals growth and positive impact.
-   **Premium Typography**: Uses `Michroma` for technical, sharp headers and `Montserrat` for clean, readable body text.

## üöÄ Key Features

-   **Global Reach**: Curated charity selections for the USA, India, and the UK.
-   **The Visionary Section**: A dedicated profile for the creator, showcasing the human mission behind the code.
-   **One-Click Sharing**: Minimalist sharing buttons with instant clipboard feedback to spread the mission.
-   **Direct Impact**: Direct routing to charity donation pages, ensuring no intermediaries or fees.
-   **Interactive Visuals**: Mission-specific 3D imagery and sleek hover animations.

## üèÅ Getting Started

### Prerequisites

-   Node.js 18.x or later
-   npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/celebrate-for-change.git
    cd celebrate-for-change
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open the app**:
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

### Deployment

The easiest way to deploy this Next.js app is via [Vercel](https://vercel.com):

1.  Push your code to GitHub.
2.  Import your repository into Vercel.
3.  Vercel will automatically detect the settings and deploy your site to a global URL.
